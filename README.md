#jQueryUI.Autocomplete.Match

[jQuery UI][] plugin for [Autocomplete Widget][] to highlight matches in dropdown lists.

```
$('input').autocomplete({
	highlight: true,
	minLength: 2,
	source: 'list.php'
});
```

 [jQuery UI]: http://jqueryui.com/
 [Autocomplete Widget]: http://jqueryui.com/autocomplete/
